Student Management System

A Student Management System is a web-based application designed to manage student records, courses, and enrollments efficiently. This project helps administrators handle student data, track course registrations, and manage fees in an organized way.

 **Features**
    Add, update, delete student details
    Manage courses (Add/Edit/Delete)
    Student enrollment in courses
    Fee management system
    Dashboard with summary (total students, courses, revenue)
    Search and pagination support
    Login authentication system
 **Technologies Used**
   Backend: Java, Spring Boot
   Frontend: HTML, CSS, Bootstrap, Thymeleaf
   Database: MySQL
   Build Tool: Maven
